# 刘津玮个人主页

这是一个基于HTML和Tailwind CSS构建的大学生个人主页，用于展示个人信息、学习成果和实践能力。

## 功能特点

- **响应式设计**：适配不同屏幕尺寸，在手机、平板和电脑上都能良好显示
- **现代化界面**：使用Tailwind CSS构建，界面美观、整洁
- **完整的信息展示**：包含个人基本信息、技能、经历、项目和联系方式等核心内容
- **交互效果**：平滑滚动、悬停动画等增强用户体验

## 目录结构

```
LiuJinWei_PersonalPage/
├── index.html          # 主页文件
└── README.md           # 项目说明
```

## 如何使用

1. 克隆或下载本仓库到本地
2. 打开 `index.html` 文件即可在浏览器中查看个人主页
3. 根据需要修改 `index.html` 文件中的个人信息

## 部署到GitHub Pages

要将个人主页部署到GitHub Pages，请按照以下步骤操作：

1. 在GitHub上创建一个新的仓库
2. 将本项目的文件上传到仓库中
3. 在仓库设置中开启GitHub Pages功能
4. 选择主分支作为源，保存设置
5. 等待几分钟后，即可通过 `https://<username>.github.io/<repository-name>` 访问个人主页

## 技术栈

- HTML5
- Tailwind CSS v3
- Font Awesome
- JavaScript

## 个性化修改

您可以通过修改 `index.html` 文件来个性化您的个人主页：

1. **基本信息**：修改姓名、学校、专业等个人信息
2. **技能**：更新专业技能和掌握程度
3. **经历**：添加或修改校园实践、实习经历等
4. **项目**：更新项目内容和描述
5. **联系方式**：修改邮箱、GitHub等联系方式
6. **样式**：通过Tailwind CSS自定义颜色、字体等样式

## 注意事项

- 请确保所有信息真实可靠
- 不要在页面中展示敏感信息，如手机号、详细地址等
- 定期更新个人主页内容，保持信息的时效性
